## 2️⃣ 동작 과정
- [ ] 다리의 길이를 입력한다.
- [ ] 입력받은 길이만큼 다리를 자동으로 생성한다.
- [ ] 게임을 종료하거나 클리어하기 전까지 반복한다.
  - [ ] 이동할 칸을 선택한다.
  - [ ] 이동할 칸이 이동할 수 있는 칸인지, 이동할 수 없는 칸인지 확인한다.
    - [ ] 다리 결과를 출력한다.
    - [ ] 이동할 수 있는 칸이면 반복한다.
    - [ ] 이동할 수 없는 칸인 경우, 게임을 재시작할지 종료할지 입력받는다.
      - [ ] 게임을 재시작할 경우, 사용자 다리의 길이를 0으로 변경한다.
      - [ ] 게임을 종료할 경우, 루프문을 빠져나간다.
- [ ] 최종 게임 결과를 출력한다.
- [ ] 게임 성공 여부를 출력한다.
- [ ] 총 시도한 횟수를 출력한다.
- [ ] 프로그램을 종료한다.
